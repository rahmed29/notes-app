# Note Taking Website

Deprecated. This really isn't that good and I now use Obsidian.MD which is almost exactly what I was aiming to create. However, I did learn a lot from this. It would my first experience working on a somewhat large personal project.

Things I learned:
- Networking basics and woring in the terminal in Ubuntu/Debian linux distros
- Important Vanilla JavaScript APIs and concepts (fetch, dom manipulation, etc)
- Basic UI-Design concepts
- Setting up web servers and creating CRUD APIs with Express JS
- Utilizing open source libraries and using NPM
- The importance of creating maintainable code
- How to use Regular expressions
- Working with MongoDB and Mongoose

Multiple times throughout the creation of this, I attempted to migrate it to React. Although it was never fully migrated. I did learn a lot of useful fundamental React concepts like using props, useState, useEffects, portals, etc

---
This is my personal note taking website made with vanilla JavaScript.
