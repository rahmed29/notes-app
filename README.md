# Note Taking Website

This is my personal note taking website made with vanilla JavaScript. It is simply an open-source, self-hosted note taking application with a minimalistic web interface for creating and organizing notes in Markdown and LaTeX. It's designed to be used with CloudFlare Access.

![Screenshot](https://raw.githubusercontent.com/rahmed29/notes-app/master/screenshot.png)

Technologies:
- Ace Editor
- Micromark
- Express JS
- MongoDB
